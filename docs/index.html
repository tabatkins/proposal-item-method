    item()

    We provide a new method, item(), that is on the prototype of the built-in indexable objects: Array, String, and TypedArrays objects. The method supports relative indexing from the end when passed a negative index.

    1 Additions to Properties of the Array Prototype Object

    1.1 Array.prototype.item ( index )

    1. Let O be ? ToObject(this value).
    2. Let len be ? LengthOfArrayLike(O).
    3. Let relativeIndex be ? ToInteger(index).
    4. If relativeIndex ≥ 0, then
      1. Let k be relativeIndex.
    5. Else,
      1. Let k be max((len + relativeIndex), 0).
    6. Return ? Get(O, ! ToString(k)).

    2 Additions to Properties of the String Prototype Object

    2.1 String.prototype.item ( index )

    1. Let O be ? RequireObjectCoercible(this value).
    2. Let S be ? ToString(O).
    3. Let len be the length of S.
    4. Let relativeIndex be ? ToInteger(index).
    5. If relativeIndex ≥ 0, then
      1. Let k be relativeIndex.
    6. Else,
      1. Let k be max((len + relativeIndex), 0).
    7. Return the String value containing the code unit at k.

    3 Additions to Properties of the %TypedArray.prototype% Object

    3.1 %TypedArray%.prototype.item ( index )

    1. Let O be the this value.
    2. Perform ? ValidateTypedArray(O).
    3. Let len be O.[[ArrayLength]].
    4. Let relativeIndex be ? ToInteger(index).
    5. If relativeIndex ≥ 0, then
      1. Let k be relativeIndex.
    6. Else,
      1. Let k be max((len + relativeIndex), 0).
    7. Return ? Get(O, ! ToString(k)).
